# Metalcore Piano Lab（中文）

一个面向金属核/前卫核风格的网页版钢琴节奏小游戏原型，核心目标是探索“从音频自动生成可玩的谱面”。本项目在浏览器端完成节拍/拍号估计、打击点提取、段落识别与谱面生成，并提供多键位与多难度的可玩体验。

## 主要功能

- 本地音频上传并自动生成谱面
- 节拍追踪与拍号估计（含 4/4、5/4、7/8 等）
- 段落识别 + 模板对齐（让重复段更一致）
- 多键位（4/6/8）与多难度
- 命中反馈与可调音效
- 深色高级感 UI + 背景 Logo

## 使用方式

1. 打开 `index.html`
2. 上传本地音频（mp3/wav/m4a）
3. 点击“分析并生成谱面”
4. 使用 D/F/J/K（或 6/8 键位）开始游戏

## 已知限制

- 该项目为轻量级浏览器原型，未使用深度学习分离音轨
- 对复杂变拍或极端节奏变化的歌曲，谱面准确度可能下降
